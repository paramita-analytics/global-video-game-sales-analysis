						Global Video Game Sales Analysis Dashboard						
												
Project Overview												
This project analyzes global video game sales data using Microsoft Excel.The objective was to identify high-performing platforms, publishers, genres, and regions through data cleaning, filtering, pivot tables, charts, and an interactive dashboard.												
												
Tools used												
1.Microsft Excel												
2.Pivot Tables												
3.Pivot Charts												
4.Slicers												
5.Data Filtering												
6.Dashboard Design												
												
Dashboard Features												
1. Total Sales KPI												
2. Total Games KPI												
3. Top Publisher KPI												
4. Top Performing Region KPI												
5. Top Platforms by Global Sales												
6. Top Publishers by Global Sales												
7. Genre-wise Global Sales Analysis
8.Regional Sales Comparison												
9. Interactive Year Slicer												
												
Key Insights												
1. Global sales reached 8,820.17 million units across 16,325 games.												
2. Nintendo emerged as the leading publisher by global sales.												
3. North America generated the highest regional sales.												
4. Action and Sports genres were among the highest-selling categories.												
5. Platforms such as PS2, X360, and PS3 contributed significantly to overall sales.												
												
Business Recommendations												
1. Focus investment on high-performing genres such as Action and Sports.												
2. Prioritize North American markets due to strong sales performance.												
3. Partner with leading publishers to improve market reach.												
4. Monitor platform trends to guide future game development strategies.
